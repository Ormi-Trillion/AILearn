# AILearn (에일리언) - 온라인 영어 학습 서비스
<img width="1320" alt="스크린샷 2024-09-09 오후 1 58 59" src="https://github.com/user-attachments/assets/e61d757f-faba-46cb-9365-f3af51d3ff25">

## 📌 프로젝트 소개

AILearn(에일리언)은 AI 기술을 활용한 혁신적인 온라인 영어 학습 플랫폼입니다. 
Alan AI를 통해 개인화된 학습 경험을 제공하며, 사용자의 영어 실력 향상을 효과적으로 지원합니다.

## 🚩 프로젝트 개요
- 프로젝트 이름: AILearn (에일리언)
- 프로젝트 기간: 2024.08.21 ~ 2024.09.12
- 배포 주소: [AILearn✨](http://your-deployment-url-here/)

## 🎯 주요 기능

1. **레벨 테스트**: 사용자의 영어 실력을 정확히 진단합니다.
2. **개인화된 학습 계획**: AI가 사용자의 레벨에 맞는 학습 계획을 제공합니다.
3. **어휘 학습**: AI 기반의 개인화된 단어장을 생성합니다.
4. **문법 및 문장 구조 학습**: AI가 생성한 예문을 통해 효과적으로 문법을 학습합니다.
5. **쓰기 연습**: AI 첨삭 기능으로 작문 실력을 향상시킵니다.
6. **게임화 요소**: 일일 챌린지와 퀘스트로 학습 동기를 부여합니다.
7. **이메일 인증**: 안전한 회원가입을 위한 이메일 인증 기능을 제공합니다.
8. **임시 비밀번호 발급**: 비밀번호 분실 시 이메일로 임시 비밀번호를 전송합니다.
