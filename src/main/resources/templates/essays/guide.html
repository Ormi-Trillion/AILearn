에세이 작성 가이드

A1~A2

목표

- 간단한 문장으로 자신의 생각 표현하기

- 기본적인 문법 구조 사용하기

- 일상적인 주제에 대해 쓰기

가이드라인

1. 길이: 50-100 단어

2. 구조: 간단한 서론, 본론, 결론 (각 1-2문장)

3. 문법: 현재 시제 위주, 간단한 과거 시제 사용

4. 어휘: 기본적인 일상 어휘 사용

5. 주제: 가족, 취미, 일상생활 등

평가 기준

- 기본적인 문장 구조의 정확성

- 간단한 아이디어 전달 능력

- 기초 어휘의 적절한 사용

B1~B2

목표

- 복잡한 문장 구조 사용하기

- 논리적인 단락 구성하기

- 다양한 주제에 대한 의견 제시하기

가이드라인

1. 길이: 150-250 단어

2. 구조: 명확한 서론, 2-3개의 본론 단락, 결론

3. 문법: 다양한 시제 사용, 조건문, 수동태 도입

4. 어휘: 주제에 맞는 특정 어휘, 동의어 활용

5. 주제: 사회 문제, 문화 비교, 환경 등

평가 기준

- 문장 구조의 다양성과 정확성

- 논리적 흐름과 단락 구성

- 주제에 대한 이해도와 분석력

C1~C2

목표

- 복잡한 아이디어를 명확하게 표현하기

- 학술적/전문적 글쓰기 기술 습득하기

- 비판적 사고와 분석 능력 보여주기

가이드라인

1. 길이: 300-500 단어

2. 구조: 체계적인 서론, 본론, 결론

3. 문법: 분사 구문, 가정법 등 사용

4. 어휘: 학술/전문 어휘, 관용구, 정교한 표현

5. 주제: 글로벌 이슈, 추상적, 학술적 주제 등

평가 기준

- 문장 구조의 다양성과 정확성

- 논리적 흐름과 단락 구성

- 주제에 대한 이해도와 분석력
